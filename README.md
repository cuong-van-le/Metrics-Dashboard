Metrics-Dashboard connects to your system to monitor metrics. It then sends this data to AWS S3 through Kinesis Firehose. This process happens continuously, allowing for near real-time updates.

### 1. Metrics Collection
The application collects various metrics from your machine, such as CPU usage, memory consumption, and network activity.

### 2. Data Streaming
Once collected, the metrics are streamed to AWS S3. This is done securely and efficiently through Kinesis Firehose.

### 3. Data Transformation
Transformations are applied to the data as it streams, preparing it for future analysis or reporting through services like AWS QuickSight.
